Coat File to NPC Coat

For creating modded NPCs. Upload a color file and it will convert it into code for a modded NPC's .meta. Does not generate a complete file, please paste in the appropriate spot in your mod file. Colors can be found at C:\Users\YOUR USERNAME\AppData\Local\Cattails_Wildwood_Story\colors. AppData is a hidden folder, press CTRL+R and enter %APPDATA% to get to it.
