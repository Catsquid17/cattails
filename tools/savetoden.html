Save File to NPC Den

For creating modded NPCs. Upload a save file and it will convert the den on that file into code for a modded den. Does not generate a complete file, please paste in the appropriate spot in your mod file. Saves can be found at C:\Users\YOUR USERNAME\AppData\Local\Cattails_Wildwood_Story\saves. AppData is a hidden folder, press CTRL+R and enter %APPDATA% to get to it.
