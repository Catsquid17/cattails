Save File to NPC Den

For editing or obtaining preexisting dens. Upload a save file and a .region file and it will replace the den from the save file with the den from the region file. Saves can be found at C:\Users\YOUR USERNAME\AppData\Local\Cattails_Wildwood_Story\saves. AppData is a hidden folder, press CTRL+R and enter %APPDATA% to get to it. Region files can be found in mods or in the gameresources folder at C:\Program Files (x86)\Steam\steamapps\common\Cattails Wildwood Story\gameresources\map
